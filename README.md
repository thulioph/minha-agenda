## Minha Agenda

Aplicação que permite a adição de contatos a uma lista qualquer. 
Forneça: **nome**, **telefone**, **e-mail** e uma **nota** para cadastrar um contato.

## Requisitos

- [Git][git]
- [NodeJS][nodejs]
- [Bower][bower]
- [MongoDB][mongodb]
- [Heroku CLI][heroku]

## Pacotes

- [Express][express]
- [AngularJS][angularjs]
- [Milligram][milligram]

- npm init
